MSH形式の仕様

・共通
bool : 1バイトの0または1
char : 1バイトの符号あり整数
int  : 2バイトの符号ありリトルエンディアン整数
real : 元の値を65536.0倍した値の4バイトの符号ありリトルエンディアン整数

・構造体

Vector3構造体
	x : real;
	y : real;
	z : real;

Triangle構造体
	v[3] : Vector3;

ColorARGB構造体
	r : real;
	g : real;
	b : real;
	a : real;

ColorRGB構造体
	r : real;
	g : real;
	b : real;

Material構造体
	face_color     : ColorARGB;
	power          : real;
	specular_color : ColorRGB;
	emissive_color : ColorRGB;

・ヘッダ
--------------------+------------+----------------+------------------------
名前                |データ型    |繰り返し        |説明
--------------------+------------+----------------+------------------------
signature           |char[4]     |1               |"MSH " MSH形式を識別する
has_material        |bool        |1               |マテリアルを含むかどうか
has_normal          |bool        |1               |法線を含むかどうか
--------------------+------------+----------------+------------------------

・マテリアル数と面数
（has_material == 0）
--------------------+------------+----------------+------------------------
名前                |データ型    |繰り返し        |説明
--------------------+------------+----------------+------------------------
face_num            |int         |1               |ファイルに含まれる面数
--------------------+------------+----------------+------------------------
（has_material == 1）
--------------------+------------+----------------+------------------------
名前                |データ型    |繰り返し        |説明
--------------------+------------+----------------+------------------------
material_num        |int         |1               |ファイルに含まれるマテリアル数
face_num            |int         |material_num    |マテリアルごとの面数
--------------------+------------+----------------+------------------------

・面情報
--------------------+------------+----------------+------------------------
名前                |データ型    |繰り返し        |説明
--------------------+------------+----------------+------------------------
face                |Triangle    |face_num        |面の情報
--------------------+------------+----------------+------------------------
has_material == 1の場合、各face_numに対して繰り返し格納される。

・マテリアル情報
--------------------+------------+----------------+------------------------
名前                |データ型    |繰り返し        |説明
--------------------+------------+----------------+------------------------
material            |Material    |material_num    |マテリアルの情報
--------------------+------------+----------------+------------------------
has_material == 0の場合、このデータセットは存在しない。

・法線情報
--------------------+------------+----------------+------------------------
名前                |データ型    |繰り返し        |説明
--------------------+------------+----------------+------------------------
normal              |Triangle    |face_num        |法線の情報
--------------------+------------+----------------+------------------------
has_normal == 0の場合、このデータセットは存在しない。
